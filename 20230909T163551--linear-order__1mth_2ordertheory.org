* Definition

#+BEGIN_DEFINITION
A /linear order/ \(L\) is a [[denote:20230909T163239][Poset]] such that
 * \(\forall x,y : L\mathbin{.} x \leq y \vee y \leq x\).
#+END_DEFINITION
